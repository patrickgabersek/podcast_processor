# 🎙️ YouTube Podcast Processor

Paste a YouTube URL → get a timestamped transcript + AI summary powered by Claude.

**Stack:** yt-dlp · faster-whisper (Whisper large-v3-turbo) · Claude Sonnet 4 · Streamlit

---

## Requirements

- Python 3.10+ — install from https://python.org
- ffmpeg — required for audio extraction
- An [Anthropic Console](https://platform.claude.com/)

---

## Setup

```bash
git clone <your-repo>
cd <your-repo>
bash setup.sh
```

The script will:
1. Check for Python and ffmpeg
2. Create a virtual environment (`.venv/`)
3. Install all dependencies from `requirements.txt`
4. Detect your GPU and install the correct torch build automatically
5. Create a `.env` file for your API key

Then open `.env` and replace `your_api_key_here` with your actual key:

```
ANTHROPIC_API_KEY=sk-ant-...
```

---

## Run

```bash
source .venv/Scripts/activate   # Windows (Git Bash)
source .venv/bin/activate       # Linux/macOS
streamlit run podcast_processor.py
```

Opens at `http://localhost:8501`

---

## How it works

```
YouTube URL
    │
    ▼
yt-dlp          — downloads best audio, converts to mp3
    │
    ▼
faster-whisper  — transcribes locally using Whisper large-v3-turbo
    │             (GPU if available, CPU fallback)
    ▼
Claude Sonnet 4 — streams a structured summary (hook, key takeaways,
                  notable quotes, actionable insights)
    │
    ▼
Streamlit UI    — timed transcript + summary, both downloadable
```

## GPU vs CPU

The setup script detects your hardware and installs the right torch version automatically.

| Hardware | Torch build | Speed estimate (1h podcast) |
|---|---|---|
| NVIDIA GPU — CUDA 12.8+ | cu128 | ~2-5 minutes |
| NVIDIA GPU — CUDA 12.4+ | cu124 | ~2-5 minutes |
| NVIDIA GPU — CUDA 12.1+ | cu121 | ~2-5 minutes |
| CPU only | CPU | ~10-20 minutes |

No GPU? It still works fine, just slower. Everything except transcription (download, Claude summary) is unaffected by hardware.

## Troubleshooting

**ffmpeg not found** — install it for your OS (see `setup.sh` output for commands).

**CUDA errors** — the app falls back to CPU automatically; no action needed.

**Download fails** — some videos are region-locked or age-restricted; yt-dlp will show the reason.

**API key error** — make sure `.env` exists and contains a valid `ANTHROPIC_API_KEY`.

**Long podcasts** — transcripts over ~80,000 characters are trimmed before being sent to Claude. The full transcript is always available to download.
